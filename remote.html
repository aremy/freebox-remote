Pour trouver le code télécommande Freebox:
  • Freebox Révolution: menu Réglages > Système > Informations Freebox Player et Server > Code télécommande réseau télécommande réseau
  • Freebox v5 (HD/Crystal): menu Paramètres > Informations générales > Code télécommande
La télécommande ne fonctionne que si vous êtes connecté à votre réseau local (Wifi ou filaire)
